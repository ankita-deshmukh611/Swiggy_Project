# Swiggy Sales Analysis Dashboard

## Project Overview
This project analyzes Swiggy order data using Excel and creates an interactive dashboard to understand sales trends and customer behavior.

## Objective
To analyze food delivery data and identify business insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning

## Key Skills
- Data Analysis
- Dashboard Creation
- Data Visualization
- Data Cleaning

## Steps Performed
1. Imported dataset
2. Cleaned missing values
3. Created pivot tables
4. Built dashboard


## Key Insights
- Highest sales categories identified
- Order trends analyzed
- Customer behavior
